# Evaluate an expression within a Record's data.

Evaluate an expression within a Record's data.

## Usage

``` r
# S3 method for class 'Record'
with(data, expr, ...)
```

## Arguments

- data:

  A Record object.

- expr:

  Expression to evaluate within the record.

- ...:

  Additional arguments (ignored)

## Value

The result of the evaluated expression.
